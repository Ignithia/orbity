Introduction

Orbity.js is a lightweight and highly customizable JavaScript library for creating interactive 3D tag clouds. It supports various shapes, touch interactions, and device orientation, making it ideal for visually engaging web applications.

Try the Playground

Features

  • Customizable shapes, including sphere, cube, pyramid, and more.
  • Touch and device orientation support for interactive experiences.
  • Dynamic tag addition, removal, and updates.
  • Configurable rotation speed, easing, and other animation settings.
  • Accessible with ARIA roles for better usability.

Getting Started

To use Orbity.js, include the library in your project:

<script src="orbity.js"></script>

Next, add a canvas element to your HTML where the tag cloud will be rendered: 

<canvas id="orbityCanvas" width="800" height="600"></canvas>

Finally, initialize Orbity.js with the following script:


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 150,
  speed: 1,
  shape: "sphere",
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
]);

Basic Example

Here is an example that demonstrates additional features of Orbity.js: 


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200, // Larger radius for a bigger tag cloud
  speed: 1, // Faster rotation speed
  shape: "cube", // Cube-shaped tag cloud
  maxVelocity: 0.2, // Maximum rotation velocity
  enableTouch: true, // Enable touch interactions
  enableOrientation: true, // Enable device orientation support
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
  { text: "CSS", color: "#2965f1" },
  { text: "React", color: "#61dafb" },
  { text: "Node.js", color: "#68a063" },
  { text: "Python", color: "#3776ab" },
  { text: "Django", color: "#092e20" },
  { text: "TypeScript", color: "#007acc" },
]);

// Dynamically add more tags
orbityInstance.addTag({ text: "Vue.js", color: "#42b883" });
orbityInstance.addTag({ text: "Angular", color: "#dd0031" });

This example creates a cube-shaped tag cloud with a larger radius, faster rotation speed, and additional tags. It also enables touch and device orientation support for interactive experiences.

Usage Examples

1. Creating a Cube-Shaped Tag Cloud

To create a tag cloud with a cube shape:


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200,
  shape: "cube",
});
orbityInstance.setTags([
  { text: "CSS", color: "#2965f1" },
  { text: "React", color: "#61dafb" },
  { text: "Node.js", color: "#68a063" },
]);

2. Adding Tags Dynamically

You can add tags dynamically after initializing the tag cloud:


orbityInstance.addTag({ text: "Python", color: "#3776ab" });
orbityInstance.addTag({ text: "Django", color: "#092e20" });

3. Updating Tag Cloud Options

To update the configuration of the tag cloud:


orbityInstance.updateOptions({
  speed: 0.005,
  shape: "pyramid",
});

Configuration Options

Below is a list of configuration options available in Orbity.js, along with their descriptions and default values:

Option Type Description Default
radius Number The radius of the tag cloud in pixels. 150
speed Number The rotation speed of the tag cloud. Higher values result in faster rotation. 0.002
shape String The shape of the tag cloud. Supported values are "sphere", "cube", "pyramid", etc. "sphere"
maxVelocity Number The maximum rotation velocity of the tag cloud. This limits how fast the cloud can spin. 0.01
enableTouch Boolean Enables touch interactions for rotating the tag cloud on touch devices. false
enableOrientation Boolean Enables device orientation support, allowing the tag cloud to respond to device tilting. false
easing String The easing function for the rotation animation. Supported values include "linear", "ease-in", "ease-out", etc. "linear"

Try It Yourself

Note: Some shapes, such as "cube" or "pyramid," require a minimum number of tags to render correctly. Ensure you add enough tags for these shapes to display properly.
