Introduction

Orbity.js is a lightweight and highly customizable JavaScript library for creating interactive 3D tag clouds. It supports various shapes, touch interactions, and device orientation, making it ideal for visually engaging web applications.

Try the Playground

Features

  • Customizable shapes, including sphere, cube, plane, torus, and more.
  • Touch and device orientation support for interactive experiences.
  • Dynamic tag addition, removal, and updates.
  • Configurable rotation speed, easing, and other animation settings.
  • Accessible with ARIA roles for better usability.
  • Hover and click effects for tags.
  • Custom fonts and styles for tags.

Getting Started

To use Orbity.js, include the library in your project:

<script src="orbity.js"></script>

Next, add a canvas element to your HTML where the tag cloud will be rendered: 

<canvas id="orbityCanvas" width="800" height="600"></canvas>

Finally, initialize Orbity.js with the following script:


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 150,
  speed: 1,
  shape: "sphere",
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
]);

Basic Example

Here is an example that demonstrates additional features of Orbity.js: 


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200, // Larger radius for a bigger tag cloud
  speed: 1, // Faster rotation speed
  shape: "cube", // Cube-shaped tag cloud
  maxVelocity: 0.2, // Maximum rotation velocity
  enableTouch: true, // Enable touch interactions
  enableOrientation: true, // Enable device orientation support
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
  { text: "CSS", color: "#2965f1" },
  { text: "React", color: "#61dafb" },
  { text: "Node.js", color: "#68a063" },
  { text: "Python", color: "#3776ab" },
  { text: "Django", color: "#092e20" },
  { text: "TypeScript", color: "#007acc" },
]);

// Dynamically add more tags
orbityInstance.addTag({ text: "Vue.js", color: "#42b883" });
orbityInstance.addTag({ text: "Angular", color: "#dd0031" });

This example creates a cube-shaped tag cloud with a larger radius, faster rotation speed, and additional tags. It also enables touch and device orientation support for interactive experiences.

Usage Examples

1. Creating a Cube-Shaped Tag Cloud

To create a tag cloud with a cube shape:


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200,
  shape: "cube",
});
orbityInstance.setTags([
  { text: "CSS", color: "#2965f1" },
  { text: "React", color: "#61dafb" },
  { text: "Node.js", color: "#68a063" },
]);

2. Creating a Torus-Shaped Tag Cloud

To create a tag cloud with a torus shape:


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200,
  shape: "torus",
  majorRadius: 150, // Distance from the center of the torus to the center of the tube
  minorRadius: 50,  // Radius of the tube
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
  { text: "CSS", color: "#2965f1" },
]);

2. Adding Tags Dynamically

You can add tags dynamically after initializing the tag cloud:


orbityInstance.addTag({ text: "Python", color: "#3776ab" });
orbityInstance.addTag({ text: "Django", color: "#092e20" });

3. Updating Tag Cloud Options

To update the configuration of the tag cloud:


orbityInstance.updateOptions({
  speed: 0.005,
  shape: "plane",
});

Hover and Click Effects

Orbity.js allows you to enable hover and click effects for tags. These effects include scaling, color changes, and opacity adjustments. 


const orbityInstance = new Orbity(canvas, {
  hoverEffect: true, // Enable hover effects
  clickEffect: true, // Enable click effects
  hoverScale: 1.2, // Scale factor for hover effect
  hoverColor: "#ffff00", // Hover color
  hoverOpacity: 1.0, // Hover opacity
});

Custom Fonts and Styles

Customize the font family and weight of tags to match your design: 


const orbityInstance = new Orbity(canvas, {
  customFont: "serif", // Font family
  customFontWeight: "bold", // Font weight
});

Configuration Options

Below is a list of configuration options available in Orbity.js, along with their descriptions and default values:

Option Type Description Default
radius Number The radius of the tag cloud in pixels. 150
speed Number The rotation speed of the tag cloud. Higher values result in faster rotation. 0.002
shape String The shape of the tag cloud. Supported values are "sphere", "cube", "plane", etc. "sphere"
maxVelocity Number The maximum rotation velocity of the tag cloud. This limits how fast the cloud can spin. 0.01
enableTouch Boolean Enables touch interactions for rotating the tag cloud on touch devices. false
enableOrientation Boolean Enables device orientation support, allowing the tag cloud to respond to device tilting. false
easing String The easing function for the rotation animation. Supported values include "linear", "ease-in", "ease-out", etc. "linear"
autoSpin Boolean Enables automatic spinning of the tag cloud when no interaction occurs. If set to false, the cloud will only move when interacted with. true
hoverEffect Boolean Enables hover effects for tags, such as scaling, color changes, and opacity adjustments. false
clickEffect Boolean Enables click effects for tags, such as scaling, color changes, and opacity adjustments. false
hoverScale Number The scale factor applied to tags when hovered over. 1.0
hoverColor String The color applied to tags when hovered over. null
hoverOpacity Number The opacity applied to tags when hovered over. 1.0
customFont String The font family used for tags. "sans-serif"
customFontWeight String The font weight used for tags. "normal"
majorRadius Number The major radius of the torus (distance from the center of the torus to the center of the tube). 150
minorRadius Number The minor radius of the torus (radius of the tube). 50

Advanced Features

Orbity.js provides advanced features for creating highly interactive and customizable 3D tag clouds. Below are some examples:

1. Easing Profiles

Orbity.js includes built-in easing profiles for smooth animations. You can choose from the following profiles:

  • Snappy: Quick ease-in and high friction for fast deceleration.
  • Smooth: Balanced ease-in and friction for natural animations.
  • Marathon: Slow ease-in and low friction for long-lasting spins.

To set an easing profile:


orbityInstance.updateOptions({
  easingProfile: "Smooth", // Choose from "Snappy", "Smooth", or "Marathon"
});

2. Custom Easing

For advanced users, Orbity.js allows you to define custom ease-in and friction values: 


orbityInstance.updateOptions({
  customEaseIn: 0.15, // Custom ease-in multiplier
  customFriction: 0.92, // Custom friction value
});

3. Auto-Easing Toggle

You can enable or disable auto-easing at any time. When disabled, the tag cloud uses raw velocities without easing: 


orbityInstance.updateOptions({
  autoEasing: false, // Disable auto-easing
});

4. Dynamic Thresholding

Orbity.js ensures a smooth stop by maintaining a minimum velocity threshold. Once the velocity falls below this threshold, it is zeroed out: 


orbityInstance.updateOptions({
  minVelocityThreshold: 0.01, // Set the minimum velocity threshold
});

5. Using Custom Shapes

Orbity.js supports custom shapes like helix, ring, verticalRing, cylinder, and torus. Here's an example of a torus-shaped tag cloud: 


const canvas = document.getElementById("orbityCanvas");
const orbityInstance = new Orbity(canvas, {
  radius: 200,
  shape: "torus",
  majorRadius: 150,
  minorRadius: 50,
});
orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e" },
  { text: "HTML", color: "#e34c26" },
  { text: "CSS", color: "#2965f1" },
  { text: "React", color: "#61dafb" },
  { text: "Node.js", color: "#68a063" },
]);

6. Pausing and Resuming Animations

Orbity.js allows you to pause and resume animations programmatically: 


orbityInstance.pause(); // Pause the animation
orbityInstance.resume(); // Resume the animation

7. Handling Events

Orbity.js supports events like tagClick, tagHover, and tagLeave. Here's an example of handling a tag click event: 


orbityInstance.on("tagClick", (tag) => {
  console.log("Tag clicked:", tag.text);
});

8. Customizing Tag Appearance

You can customize the appearance of individual tags by specifying properties like fontSize and color: 


orbityInstance.setTags([
  { text: "JavaScript", color: "#f7df1e", fontSize: 20 },
  { text: "HTML", color: "#e34c26", fontSize: 18 },
  { text: "CSS", color: "#2965f1", fontSize: 16 },
]);

9. Destroying the Instance

To clean up and remove all event listeners, you can destroy the Orbity instance: 


orbityInstance.destroy();

10. Controlling Automatic Spinning

The autoSpin option allows you to enable or disable automatic spinning of the tag cloud. When set to false, the cloud will only move when interacted with. 


const orbityInstance = new Orbity(canvas, {
  radius: 150,
  autoSpin: false, // Disable automatic spinning
});

This example disables automatic spinning, requiring user interaction to move the tag cloud.

Managing Tags Dynamically

Orbity.js allows you to add, remove, or update tags dynamically after the tag cloud has been initialized.

1. Adding Tags

Use the addTag method to add new tags:


orbityInstance.addTag({ text: "Vue.js", color: "#42b883" });
orbityInstance.addTag({ text: "Angular", color: "#dd0031" });

2. Removing Tags

Use the removeTag method to remove specific tags by their text: 


orbityInstance.removeTag("Vue.js");

3. Updating Tags

Use the updateTag method to modify an existing tag:


orbityInstance.updateTag("Angular", { color: "#c3002f" });

Event Hooks & Listeners

Orbity.js provides a flexible event system that lets you add or remove listeners for tag interactions and other events. Use the on and off methods to register or remove event handlers.

Supported Events

  • tagClick: Fired when a tag is clicked.
  • tagHover: Fired when a tag is hovered.
  • tagLeave: Fired when the mouse leaves a tag.

Registering an Event Listener


// Listen for a tag click
orbityInstance.on("tagClick", (tag, index, event) => {
  alert(`You clicked on ${tag.text}`);
});

// Listen for tag hover
orbityInstance.on("tagHover", (tag, index, event) => {
  console.log("Tag hovered:", tag.text);
});

// Listen for tag leave
orbityInstance.on("tagLeave", (tag, index, event) => {
  console.log("Mouse left tag:", tag.text);
});

Removing Event Listeners

To remove an event listener, use the off method. You can remove all listeners for an event, or just a specific callback: 


// Remove all tagClick listeners
orbityInstance.off("tagClick");

// Remove a specific callback
function onClick(tag) { /* ... */ }
orbityInstance.on("tagClick", onClick);
orbityInstance.off("tagClick", onClick);

Custom Interactivity

With event hooks, you can easily add tooltips, pop-ups, or any custom UI when a tag is clicked or hovered. This makes Orbity highly extensible for interactive web experiences.
